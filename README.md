<hr><br><h2>Conventional Commits</h2>

</div>


###### Structured

```
<type>[optional scope]: <description>
[optional body]
[optional footer(s)]
```

###### Type

```
fix:
feat:
BREAKING CHANGE:
build:
chore:
ci:
docs:
style:
refactor:
perf:
test:
```

##### Example

```
feat(api)!: send an email to the customer when a product is shipped
```
